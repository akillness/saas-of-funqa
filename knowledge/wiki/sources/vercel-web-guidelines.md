# Vercel Web Guidelines

## Source

- URL: https://raw.githubusercontent.com/vercel-labs/web-interface-guidelines/main/command.md

## Notes

- 아이콘 버튼은 `aria-label` 필요
- 포커스 가시성 대체 없이 `outline-none` 금지
- 상태는 URL에 반영
- 폼은 label, autocomplete, inline error를 갖춰야 함
- 애니메이션은 `prefers-reduced-motion` 고려 필요
